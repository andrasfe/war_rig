# PSBPAUTL

**Type:** OTHER (UTILITY)

## Purpose

This file defines a Program Specification Block (PSB) for IMS database access. It specifies the database (DBPAUTP0), processing options, and sensitive segments (PAUTSUM0, PAUTDTL1) that the program is authorized to access. The PSB is generated using the ASSEM language.
